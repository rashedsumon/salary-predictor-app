# Machine Learning Engineer Salary Predictor (2024 Dataset)

An interactive Streamlit web application running a machine learning regression pipeline to predict human capital valuation across data professions based on 2024 metrics.

## Setup Instructions

### 1. Local Run
Ensure you have Python installed, then follow these commands inside your project root:
```bash
# Install target dependencies
pip install -r requirements.txt

# Start the Streamlit application interface
streamlit run streamlit_app.py
